Ping - утилита, которая позволяет проверить доступность хоста через протокол ICMP.
	Решает следующие задачи:
1. Проверка доступности хоста.
	Например, представим ситуацию, в которой к хосту невозможно подключиться по ssh. Утилита ping в данном случае позволит исключить или потвердить факт недоступности машины.
2. Опция -f позволяет позволяет выполнить нагрузочное тестирование канала или даже заглушить интернет соединение на одной из машин.
	Так как в этом режиме пакеты передаются без задержек, то можно, например, совершить контролируемую ddos-атаку, чтобы выработать механизм противодействия угрозе такого рода.


Traceroute - утилита, позволяющая узнать, через какие узлы проходит пакет прежде чем он окажется на узле - получателе.
	Решает следующие задачи:
1. Используется для обнаружения обрыва в цепочке маршрутизаторов.
2. Можно применить для исследования сети. Например, можно попытатья определить использование подсетей провайдером.


Nslookup - утилита, предоставляющая пользователю интерфейс командной строки для обращения к системе DNS.
	Решает следующие задачи:
1. Проверка работоспособности DNS.
2. Вывод ip адреса узла сети по его доменному имени и наоборот.
3. Определение серверов имён, которое использует домен.



Dig - многофункциональный инструмент для опроса DNS-серверов.
	Решает следующие задачи:
1. Получение информации о о различных записях DNS, включая адреса хостов, почтовые обмены и серверы имен.
2. Позволяет проводить опрос группы доменов, указаных в файле посредством опции -f.


Ss - это инструмент, который используется для отображения информации о сетевых сокетах в системе Linux.
	Решает следующие задачи:
1. Вывод всех установленных соединений.
2. Фильтрация вывода списка всех сокетов по определённым параметрам. Например вывести только tcp сокеты, или сокеты ожидающие подключения.


Nmap - инструмент для исследования сети и проверки безопасности.
	Решает следующие задачи:
1. Может быть использован для проверки безопасности.
2. Определения сервисов запущенных на узле.
3. Идентификации ОС и приложений.
4. Определения типа фаерволла используемого на сканируемом узле.
